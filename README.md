# 📦 Inventory Management System

A console-based **Inventory Management System** developed in **C** that allows users to efficiently manage inventory through CRUD operations, item search, expiry date tracking, and stock reporting. The project demonstrates the use of core C programming concepts such as structures, unions, arrays, functions, and string handling.

---

## 🚀 Features

- ➕ Add new inventory items
- ✏️ Update existing item details
- ❌ Delete items from inventory
- 🔍 Search items by name
- 📋 Display all inventory items
- 📅 Track expired items
- 📊 Generate stock reports

## ⚙️ How to Run

### Compile

```bash

```

### Run

```bash
./inventory
```

> **Windows (MinGW):**

```bash
inventory.exe
```
